# DotNetprojet2

Pour éxecuter l'application et accéder au site web, voicis les étapes a suivre :

- Double cliquez sur "P2FixAnAppDotNetCode.sln" pour l'ouvrir avec Visual Studio
- Selectionnez IIS Express en temps que mode d'execution
- Cliquez sur IIS Express, Visual Studio va alors compiler le projet et lancer l'application.
